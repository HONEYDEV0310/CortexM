# STM32 및 ESP32 기반 스마트 자동 타겟팅 터렛 (Smart Auto-Targeting Turret)

> **STM32F429**와 **ESP32**를 기반으로 한 하드웨어 임베디드 시스템이자, 카메라 비전 인식 및 무선 통신을 활용한 **실시간 자동 타겟팅 2축 짐벌(Pan-Tilt Turret) 프로젝트**입니다. 
> 실제 하드웨어 제작 전, 제어 알고리즘 검증과 PID 오토튜닝(Auto-Tuning)을 사전에 수행하기 위해 **PyBullet 물리 엔진 시뮬레이션 환경**을 함께 구축하여 개발하고 있습니다.

---

## 🎯 프로젝트 개요 (Project Overview)

본 프로젝트는 임베디드 하드웨어 제어와 컴퓨터 비전, 무선 통신 기술을 융합하여 지정된 타겟을 자동으로 조준하고 추적하는 스마트 터렛 시스템을 구현하는 것을 목표로 합니다. 

실제 MCU 환경에 펌웨어를 올리기 전, 가상 공간(PyBullet)에서 2축 짐벌의 기구학적 거동을 테스트하고 **PID 제어 파라미터를 시뮬레이션 상에서 미리 튜닝**함으로써, 실물 하드웨어 구동 시 발생할 수 있는 시행착오와 리스크를 최소화합니다.

---

## 🛠️ 시스템 아키텍처 및 하드웨어 스택 (System Architecture)

* **메인 MCU 1:** **STM32F429** (고속 모터 제어, 2축 짐벌 PWM 구동, 센서 데이터 처리)
* **메인 MCU 2:** **ESP32** (Wi-Fi/Bluetooth 무선 통신, 카메라 비전 데이터 송수신 및 상위 제어 연동)
* **액추에이터 (Actuator):** 2-DOF Pan-Tilt 서보/스테퍼 모터 메커니즘
* **비전 및 센서 (Vision & Sensor):** 타겟 인식을 위한 카메라 센서 모듈
* **시뮬레이션 및 튜닝 환경 (Simulation):** 
  * 언어: Python 3.8+
  * 물리 엔진: **PyBullet** (다물체 짐벌 동역학 모사, PID 제어기 사전 검증 및 오토튜닝)
